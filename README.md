# LuaLaTeX_QuizSheet
より簡単に美しい自作問題集がLuaLaTeXで作成できるアシストツール。
詳細はReadme.pdfに記載。

## クラウドサービスを用いたコンパイルについて
### Cloud LaTeX
利用したいプロジェクトにて`quizsheet.sty`をアップロードし、texファイルと同じ階層に置くことで利用可能。

（公式HP ヘルプ（よくあるご質問）[ https://cloudlatex.io/help#original_sty ]より一部改変して引用。）

なお、コンパイル時に大量の警告があるが、表のサイズに対する警告であるため無視して構わない。
